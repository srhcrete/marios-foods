# README
1. `$ git clone` [this repository](https://github.com/srhcrete/marios-foods.git)
2. `$ cd marios_food`
3. `$ bundle install`
4. `$ rake db:create`
5. `$ rake db:migrate`
6. `$ rake db:test:prepare` (for testing environment)
7. `$ rake db:reset`
8. `$ rails s` _to start the server, then navigate to localhost:3000 in your browser._
